# 📅 1895 年：歷史的轉折點

這一年，東亞的命運被重新改寫。

<div style={{border: '1px solid #ddd', padding: '20px', borderRadius: '10px'}}>

<Tabs>
  <TabItem value="taiwan" label="🇹🇼 台灣史" default>
    
    ### 😢 乙未割臺
    * **馬關條約**：清廷將台灣割讓給日本。
    * **台灣民主國**：唐景頌等人成立短暫政權抗日。
    * **乙未戰爭**：客家義軍（如姜紹祖）與各地民兵抵抗日軍接收。

  </TabItem>
  <TabItem value="china" label="🇨🇳 中國史">
    
    ### 📉 甲午戰敗
    * **北洋水師覆沒**：洋務運動宣告失敗。
    * **李鴻章赴日**：簽訂屈辱的馬關條約。
    * **公車上書**：康有為、梁啟超發起，維新變法的前奏。

  </TabItem>
  <TabItem value="world" label="🌍 世界史">
    
    ### 🧪 科學與電影
    * **X光發現**：倫琴發現X射線，獲得諾貝爾獎。
    * **電影誕生**：盧米埃兄弟在巴黎播放了第一部電影《火車進站》。
    * **無線電**：馬可尼開始進行無線電波實驗。

  </TabItem>
</Tabs>
